# Assets Directory

This directory contains the app assets for Crave Kitchen.

## Required Files

- `icon.png` - App icon (1024x1024 PNG)
- `splash.png` - Splash screen image (1242x2436 PNG)
- `adaptive-icon.png` - Android adaptive icon (1024x1024 PNG)
- `favicon.png` - Web favicon (48x48 PNG)

## Placeholder Assets

- **Splash Screen**: 1242x2436 pixels, PNG format
- **Adaptive Icon**: 1024x1024 pixels, PNG format
